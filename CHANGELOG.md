# Changelog

All notable changes to Prompt Maker will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2025-10-22

### Changed
- Locked Anthropic connector to claude-3-7-sonnet-20250219
- Removed retired candidate claude-3-5-sonnet-20240620
- Added resolvedModel metadata and enhanced 404 logging
- Created sample export for Template Library

### Added
- WordPress integration adapter and import guide
- Anthropic connector configuration with accuracy parameters
- Heartbeat logging for daily health checks
- Release bundle creation workflow

## [1.0.0] - 2025-10-19

### Added
- Initial stable release
- Three output modes: Prompt Template, Prompt Engineer Agent, Media Prompt Blueprint
- Trust & Safety scoring system
- Mock generation fallback
- In-memory storage with extensible interface
